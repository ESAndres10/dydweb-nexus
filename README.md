# DYDWEB Nexus

Sitio web corporativo de DYDWEB Nexus construido con Next.js, TypeScript y Tailwind CSS.

## Scripts

```bash
npm install
npm run lint
npm run typecheck
npm run build
npm run start
```

## Variables de entorno

```bash
NEXT_PUBLIC_SITE_URL=https://dydweb.co
GOOGLE_SHEETS_WEBHOOK_URL=
```

`GOOGLE_SHEETS_WEBHOOK_URL` debe configurarse en producción para guardar los formularios en Google Sheets.
